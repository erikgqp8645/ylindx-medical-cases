# 广义伤寒全景索引(Three-View Cross-Reference)

> 与 references/broad_typhoid_nine_grid.md(视角3·九宫格)、
>   references/broad_typhoid_five_categories.md(视角1·《难经》五分类)、
>   references/broad_typhoid_six_channels.md(视角2·六经传变)
>   形成三视角交叉引用索引。
>
> 本文件是「广义伤寒分类」三视角的**总览+交叉索引**。
> 一份外感病医案,通常需要**三视角联合定位**才能完整描述:
> - **九宫格**:确定病位(表/半/里) + 病性(寒/热/湿)
> - **五分类**:确定病因(风/寒/湿/热/温)+ 经典方
> - **六经**:确定经层(太阳/阳明/少阳/三阴)+ 传变路径

## 一、三视角的整体对照

```
            九宫格(病位×病性)        五分类(病因+病性)        六经(经层+传变)
                ↓                          ↓                          ↓
            9 格 + 跨格            中风/伤寒/湿温/热病/温病    太阳→阳明→少阳→三阴
                                    + 卫气营血对接              + 合病/并病/坏病
                                    + 三焦对接

        描述「病在哪 + 是什么病性」  描述「是什么病」           描述「病在哪条经」
        适合临床辨证                 适合理论溯源                适合传变分析
```

## 二、医林独啸斋三视角覆盖统计(456篇)

```
视角           命中数        占总库比
九宫格(病位×病性)  443 篇      97%  ← 临床最广
六经(经层)        414 篇      91%
五分类(病因)      301 篇      66%

三视角同时命中    292 篇      64%  ← 绝大多数外感病医案
```

> 临床实际:**64% 的外感病医案需要三视角联合定位**。
> 九宫格覆盖最广(临床辨证最常用),
> 六经次之(经方体系最常用),
> 五分类相对窄(更适合病因追溯)。

## 三、三视角交叉对照表

```
              五分类           九宫格(病位)        六经(经层)
中风         (表,寒)        太阳病
伤寒(狭义)   (表,寒)        太阳病
湿温         (表/半/里,湿)   太阳/少阳/太阴(湿温传变较慢)
热病         (里,热)        阳明病
温病         (表/里,热)     太阳/阳明/少阴/厥阴(卫气营血)
```

**关键洞察:**
- 中风与伤寒:病位都在「表-寒」,都在「太阳病」——五分类区分病因(风/寒),九宫格统称表寒,六经统称太阳。
- 湿温:跨病位(表/半/里皆有),跨经(太阳→少阳→太阴)——反映湿邪黏腻、缠绵传变的特性。
- 热病:聚焦「里-热」与「阳明」——气分大热是阳明主证。
- 温病:跨度最大,可贯穿六经——卫气营血是更细的病程轴。

## 八、三视角联合定位的临床实战模板
──────────────────────────────────────────────────

当一份新医案来时,按以下三步走:

```
步骤 1:九宫格定位(病位×病性)
   ├─ 病位:表 / 半 / 里?
   ├─ 病性:寒 / 热 / 湿?
   └─ 组合:9 格之一 / 跨格

步骤 2:五分类定位(病因+病性)
   ├─ 中风(汗出恶风)?
   ├─ 伤寒(无汗恶寒)?
   ├─ 湿温(缠绵不愈)?
   ├─ 热病(大热大渴)?
   └─ 温病(发热重恶寒轻)?

步骤 3:六经定位(经层+传变)
   ├─ 经层:太阳/阳明/少阳/太阴/少阴/厥阴?
   ├─ 传变:传/直中/合病/并病/坏病?
   └─ 趋势:向深传?向浅出?
```

### 实战例:

**医案:** 4岁小孩高烧39.5℃ 1月余,退烧药当饭吃,麻附辛一剂治愈

```
步骤 1(九宫格):
   - 病位:表 + 里(外寒+内阳虚)
   - 病性:寒(无汗)
   - 组合:跨格(表寒+里寒)

步骤 2(五分类):
   - 主证:伤寒狭(无汗)+中风底子(可能反复感冒)
   - 兼:湿温不明显
   - 病程长,正虚

步骤 3(六经):
   - 主经:太阳(表寒)+少阴(底子)
   - 传变:太少两感(直中少阴)
   - 趋势:阳虚底子,直入少阴

综合:太少两感,温阳解表,麻附辛汤。
```

## 一句话核心心法

「广义伤寒全景索引的核心价值,在于揭示一个反直觉的事实:**64% 的外感病医案需要三视角联合定位**——单一视角都有局限。九宫格告诉你病位和病性,五分类告诉你病因和病种,六经告诉你经层和传变;三视角互为经纬,才能把一份医案完整装入「病在何位 + 是什么病 + 在哪条经 + 如何传变」的四维坐标。临床实战时,九宫格是入口(最快定位病位病性),六经是过程(传变与合病分析),五分类是出口(溯源至经典方剂)。三视角配合使用,就是外感病辨证论治的「北斗七星」。」
