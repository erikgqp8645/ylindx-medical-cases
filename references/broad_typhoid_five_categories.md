# 广义伤寒分类·视角1·经典五分类(《难经》式)

> 与 references/broad_typhoid_nine_grid.md(视角3·九宫格)、
>   references/broad_typhoid_six_channels.md(视角2·六经) 形成三视角。
>
> 本文件按《难经·五十八难》「伤寒有五」分类:
> 中风 / 伤寒 / 湿温 / 热病 / 温病。
>
> 这是中医外感病最古老的分类范式,以**病因+病性**为主轴。
> 现代临床中,五类常以复合形式出现(合病/并病/寒温并用/温病伏邪),
> 单类病机反而少见。

## 一、《难经》五分类的原始定义

> 「伤寒有几?其脉有变不?
> 然:伤寒有五,
>  有中风,有伤寒,有湿温,有热病,有温病。
>  所苦各不同。」
> ——《难经·五十八难》

**五类核心鉴别表:**

| 类别 | 病因 | 病性 | 典型脉 | 经典方 |
|---|---|---|---|---|
| 中风 | 风邪 | 表虚 | 浮缓 | 桂枝汤 |
| 伤寒(狭义) | 寒邪 | 表实 | 浮紧 | 麻黄汤 |
| 湿温 | 湿热 | 黏腻缠绵 | 濡缓 | 三仁汤 |
| 热病 | 热邪 | 气分大热 | 洪大 | 白虎/承气 |
| 温病 | 温邪 | 温邪上受 | 浮数 | 银翘/桑菊 |

**关键提醒:**
- 此处「伤寒」是**狭义**,指寒邪束表的麻黄汤证;
  而「伤寒有五」的「伤寒」是**广义**,指一切外感热病。
- 「中风」不是今天所说的脑血管中风,而是「伤于风」之义,等同桂枝汤证。
- 「热病」与「温病」在古代文献中常混用,广义上同义,狭义上:
  「热病」偏气分大热,「温病」偏温邪上受。

## 三、五分类逐类详解(每类包含:诊断要点+经典方+医案)

### (1) 中风(桂枝汤证) — 253 篇

**经典出处:** 《伤寒论》第 2 条:「太阳病,发热,汗出,恶风,脉缓者,名为中风。」

**核心病机:**
- 风邪袭表、营卫不和
- 表虚证:汗出、恶风、脉浮缓
- 卫强营弱、营阴外泄

**辨证要点:**
- 发热 + 汗出 + 恶风 + 脉浮缓
- 鼻鸣干呕、啬啬恶寒、淅淅恶风
- 太阳病提纲证(脉浮、头项强痛、恶寒)+ 汗出 = 中风

**代表方剂:**
- 桂枝汤(主方):桂枝、白芍、生姜、大枣、甘草
- 桂枝加厚朴杏子汤(兼咳喘)
- 桂枝加葛根汤(兼项强)
- 桂枝新加汤(兼身痛脉沉迟)
- 桂枝加附子汤(兼阳虚汗漏)
- 桂枝加龙骨牡蛎汤(兼烦惊)
- 桂枝加芍药汤(兼腹满痛)

**医林独啸斋典型案例:**

| 案例 | 主方 | 关键病机 |
|---|---|---|
| 5岁小儿荨麻疹全身起风团奇痒无比 | 桂枝加荆防白 | 表虚+痒疹 |
| 一家三人外感咳嗽不止,病机却各不同 | 桂枝加厚朴杏子 | 表虚+咳喘 |
| 久治不愈的更年期综合征三诊而痊 | 桂枝加龙骨牡蛎 | 表虚+烦惊 |
| 一剂苓桂术甘汤下去手足回温 | 桂枝+苓术 | 表虚+水饮 |
| 二阳后头晕目眩不能上学,补中益气两天治愈 | 桂枝+补中益气 | 表虚+气陷 |

**经典医理金句:**
> "中风表虚证,有汗恶风脉浮缓,用桂枝汤调和营卫。
> 桂枝辛温解肌、白芍酸收敛阴,二者相合则调和营卫。
> 生姜助桂枝散邪、大枣助白芍养阴。
> 炙甘草调和诸药,兼与桂枝辛甘化阳、与白芍酸甘化阴。"
> ——医学共识

### (2) 伤寒(狭义,麻黄汤证) — 172 篇

**经典出处:** 《伤寒论》第 3 条:「太阳病,或已发热,或未发热,必恶寒,体痛,呕逆,脉阴阳俱紧者,名为伤寒。」

**核心病机:**
- 寒邪束表、卫阳被遏
- 表实证:无汗、恶寒、脉浮紧
- 寒性收引、腠理闭塞

**辨证要点:**
- 发热 + 无汗 + 恶寒 + 脉浮紧
- 头痛、身痛、骨节疼痛
- 一身尽痛、喘息、呕逆
- 太阳病提纲证 + 无汗 = 伤寒

**代表方剂:**
- 麻黄汤(主方):麻黄、桂枝、杏仁、甘草
- 麻黄加术汤(兼湿)
- 麻杏苡甘汤(兼湿热)
- 大青龙汤(兼里热烦躁)
- 三拗汤(轻证,无桂枝)
- 麻附辛汤(太少两感)

**医林独啸斋典型案例:**

| 案例 | 主方 | 关键病机 |
|---|---|---|
| 4岁小孩高烧39.5月余退烧药当饭吃,麻附辛一剂治愈 | 麻附辛 | 表实+少阴阳虚 |
| 一岁小孩高烧39.2伴抽动,大青龙加附子汤半剂速效 | 大青龙+附子 | 表实+里热+阳虚 |
| 大青龙加附子汤流感瘟疫高烧表寒未退肾阳先衰胃热尚炽 | 大青龙+附子 | 表实+阳虚+里热 |
| 葛根汤专题痛经子宫内膜增厚3剂大减9剂愈 | 葛根汤 | 表实+经期 |
| 宝宝发烧流黄稠涕剧烈咳嗽腹灼热而身恶寒 | 麻杏石甘 | 表实+里热+咳 |
| 小儿甲流后两腿疼痛无力走不动路,麻杏苡甘汤一剂而愈 | 麻杏苡甘 | 表实+湿热 |
| 小孩咽喉化脓反复高烧40.5五天上吐下泻 | 葛根+麻黄 | 表实+里热+腹泻 |

**经典医理金句:**
> "麻黄汤是辛温发汗的代表方,麻黄开腠理、桂枝通阳、杏仁宣肺、甘草调和。
> 麻黄配桂枝,发汗力强;
> 麻黄配石膏(大青龙),发汗兼清里热。
> 麻黄配附子(麻附辛),发汗兼温阳——太少两感必用之方。"
> ——医学共识

### (3) 湿温 — 143 篇

**经典出处:** 《温病条辨》上焦篇:「头痛恶寒,身重疼痛,舌白不渴,脉弦细而濡,面色淡黄,胸闷不饥,午后身热,状若阴虚,病难速已,名曰湿温。」

**核心病机:**
- 湿热困阻三焦、缠绵难愈
- 黏腻之邪、如油入面
- 热为湿遏、湿蒙热郁

**辨证要点:**
- 身热不扬(发热但皮肤不觉烫)
- 头重如裹、身重乏力
- 胸闷脘痞、不饥不食
- 苔腻(白腻/黄腻)、脉濡缓或濡数
- 午后热甚、缠绵不愈

**代表方剂:**
- 三仁汤(主方):杏仁、白蔻仁、薏苡仁
- 达原饮(湿阻膜原):草果、厚朴、槟榔
- 甘露消毒丹(湿热并重):滑石、茵陈、黄芩
- 藿朴夏苓汤(湿重于热)
- 宣痹汤(湿热痹证)
- 三仁汤合苍术白术(湿温后期,中焦)

**医林独啸斋典型案例:**

| 案例 | 主方 | 关键病机 |
|---|---|---|
| 新冠复阳发热20余天伴肺炎:经方思维疗愈湿温医案 | 柴桂三仁+苍术 | 湿温三诊 |
| 吃退烧药输液反令发烧更严重,原因在于湿邪 | 三仁汤 | 湿温误治 |
| 76岁老人大白肺75%肺受累,柴苓达原饮一剂即显效 | 柴苓达原 | 湿热+肺实变 |
| 反复发烧服退烧药汗出热不退,先寒后热原因常为湿邪 | 达原饮 | 湿温前驱 |
| 舌象定乾坤:咳嗽少痰声音嘶哑口干饮水不解,大力化湿一剂速愈 | 化湿方 | 湿温干咳 |
| 二加柴苓汤新冠甲乙流支原体肺炎疫情中速愈 | 二加柴苓 | 湿温+半表半里 |
| 反复发烧六天乍寒乍热浑身不舒服 | 达原饮 | 湿温膜原 |
| 儿童湿温+气虚:人参败毒散+三仁 | 人参败毒 | 湿温+虚人 |

**核心医理(南极师):**
> "湿温病的湿邪,黏腻绵延、如油入面;
> 大家回忆回忆——吃火锅的时候,油溅到衣服上了是不是特别难以清洗干净?
> 是不是洗了好久还会留个印子?
> 我很多好衣服都这样弄脏过,甚是恼人。
> 而古人形容湿温病是'如油入面、难分难解'。"
> ——新冠复阳发热20余天伴肺炎

> "临床上的湿温病绝大多数时候都不能速愈,这是客观规律不可违抗。
> 诸如麻黄汤证,往往一剂即痊。
> 而三仁汤证、甘露消毒丹证的患者往往需要连续服药一两周才能痊愈。
> 何故?湿邪黏腻绵延之故也。"
> ——新冠复阳发热20余天伴肺炎

> "治疗湿温病绝对不是'禁汗、禁下、禁止滋阴'的,
> 说什么湿温病不能用桂枝麻黄的人,
> 根本没有理解仲景随证治之的思想。
> 我治疗湿温病,从来都是该解表的时候就不忌讳麻桂。"
> ——新冠复阳发热20余天伴肺炎

### (4) 热病(阳明气分大热) — 203 篇

**经典出处:** 《伤寒论》第 176 条:「伤寒,脉浮滑,此表有热,里有寒,白虎汤主之。」

**核心病机:**
- 阳明气分大热(无形热邪弥漫)
- 大热、大汗、大渴、脉洪大(白虎汤「四大」)
- 或热结腑实、痞满燥实(承气汤「五实」)

**辨证要点:**
- 阳明病提纲:「阳明之为病,胃家实是也。」
- 外证:身热、汗自出、不恶寒反恶热
- 白虎汤证:大热大汗大渴脉洪大
- 承气汤证:痞满燥实、谵语、潮热、腹满痛
- 气分证:舌红苔黄、脉数、高热不退

**代表方剂:**
- 白虎汤(气分热):石膏、知母、粳米、甘草
- 白虎加人参汤(兼伤津):加人参
- 白虎加桂枝汤(兼表寒):加桂枝
- 白虎加苍术汤(兼湿):加苍术
- 大承气汤(腑实重证):大黄、芒硝、厚朴、枳实
- 小承气汤(腑实轻证):大黄、厚朴、枳实
- 调胃承气汤(腑实缓下):大黄、芒硝、甘草
- 增液承气汤(兼阴虚):加玄参、麦冬、生地
- 宣白承气汤(肺腑合病):加杏仁、瓜蒌

**医林独啸斋典型案例:**

| 案例 | 主方 | 关键病机 |
|---|---|---|
| 五根汤方证:高烧40℃以上亦可一剂退热 | 五根汤 | 透热神方(白虎升级) |
| 70岁老人大叶性肺炎肠梗阻反复高烧40说胡话 | 白虎+承气 | 气分+腑实 |
| 低烧数月倦怠嗜睡头晕头痛求医无数均无效,经方一剂而痊 | 白虎+经方 | 长期气分余热 |
| 流感发烧38℃上吐下泻一天无法进食全身酸痛 | 大柴胡+承气 | 少阳+阳明 |
| 一家四口感染新冠(柴胡+白虎+承气合用) | 三阳合治 | 气分大热 |
| 80岁老人慢阻肺心力衰竭冠心病发烧38.4 | 白虎+真武 | 气分+少阴 |
| 二阳发烧38腹泻燥热,柴葛芩连汤一剂速愈 | 葛根芩连 | 阳明+湿热 |

**经典医理:**
> "阳明气分热,用白虎汤辛寒清热。
> 白虎汤并非四大症悉具才用,只要是气分大热、舌红苔黄、脉数,即可考虑。
> 腑实已成,用承气汤苦寒攻下。
> 白虎、承气是阳明病的两个主要治法,前者清无形之热,后者下有形之实。"
> ——医学共识

### (5) 温病(温邪上受) — 261 篇(覆盖最高)

**经典出处:** 《温热论》叶天士:「温邪上受,首先犯肺,逆传心包。」

**核心病机:**
- 温邪从口鼻而入,首先侵犯肺卫
- 卫→气→营→血,逐步深入
- 或逆传心包(直入营血)
- 风温、春温、暑温、湿温、秋燥、冬温皆属之

**辨证要点:**
- 发热重、恶寒轻、口渴、咽痛
- 舌边尖红、苔薄白/薄黄
- 脉浮数
- 卫→气→营→血传变规律
- 三焦传变:上焦→中焦→下焦

**代表方剂:**

| 卫气营血阶段 | 代表方 |
|---|---|
| 卫分(辛凉解表) | 银翘散、桑菊饮 |
| 气分(辛寒清热/苦寒攻下) | 白虎、承气 |
| 营分(清营透热) | 清营汤 |
| 血分(凉血散血) | 犀角地黄汤、化斑汤 |

| 三焦部位 | 代表方 |
|---|---|
| 上焦(肺) | 银翘散、桑菊饮、麻杏石甘 |
| 中焦(胃/肠) | 白虎、承气 |
| 下焦(肝肾) | 加减复脉汤、三甲复脉汤 |

| 温病病种 | 代表方 |
|---|---|
| 风温 | 银翘散、桑菊饮 |
| 春温(伏温) | 蒿芩清胆汤 |
| 暑温 | 白虎加人参、新加香薷饮 |
| 湿温 | 三仁汤、达原饮 |
| 秋燥 | 桑杏汤、沙参麦冬汤 |
| 冬温 | 银翘散加减 |
| 温毒 | 普济消毒饮、五味消毒饮 |

| 调胃桑菊饮:小孩因肺热而发烧多日不退咳嗽不止 | 风温 | 桑菊饮 |
| 高烧39三天刀片嗓咽喉红肿:葱豉升降散一剂热退 | 风温 | 葱豉+升降散 |

## 四、五分类的复合病机(临床真实图景)
──────────────────────────────────────────────────

「伤寒有五」是理论分类,但临床所见,五类常以复合形式出现。
这一节把跨类的常见病机总结出来。

### 复合[1] 中风 + 温病 = 表虚 + 温邪(桂枝+银翘合用)

**病机:** 表虚底子+外感温邪
**代表案例:** 5岁小儿荨麻疹(桂枝+银翘思路)

### 复合[2] 伤寒 + 热病 = 表寒 + 里热(大青龙/麻杏石甘)

**病机:** 寒束于表、热郁于里
**代表案例:**
- 大青龙加附子汤(流感瘟疫)
- 宝宝发烧流黄稠涕剧烈咳嗽(麻杏石甘)
- 一岁小孩高烧39.2伴抽动(大青龙+附子)
- 92岁老人五天五夜不得眠(寒包火)

### 复合[3] 伤寒 + 湿温 = 表寒 + 湿(麻黄加术/麻杏苡甘)

**病机:** 寒湿在表
**代表案例:**
- 小儿甲流后两腿疼痛无力,麻杏苡甘汤
- 反复发烧服退烧药汗出热不退(湿+寒)

### 复合[4] 温病 + 湿温 = 温邪 + 湿阻(三仁+银翘合方)

**病机:** 温邪夹湿
**代表案例:**
- 二加柴苓汤系列
- 调胃桑菊饮(肺热+湿)

### 复合[5] 中风 + 阳虚 = 表虚 + 少阴(桂枝加附子)

**病机:** 太少合病轻证
**代表案例:**
- 一家三人二阳均用经方一剂速愈
- 哺乳期感冒+阳虚

### 复合[6] 伤寒 + 阳虚 = 表实 + 少阴(麻附辛)

**病机:** 太少两感
**代表案例:**
- 4岁小孩高烧39.5月余(麻附辛)
- 8岁小儿二阳发烧疲劳嗜睡

### 复合[7] 热病 + 湿温 = 气分热 + 湿(白虎加苍术)

**病机:** 阳明热+太阴湿
**代表案例:** 6月首阳高烧39伴刀片嗓(葛根+达原+白虎)

### 复合[8] 中风 + 伤寒 = 麻桂各半(桂枝麻黄各半汤)

**病机:** 表虚表实错杂
**代表案例:** 小儿反复高烧系列

### 复合[9] 温病 + 阳虚 = 温病 + 少阴(扶阳透邪)

**病机:** 阳虚底子外感温邪
**代表案例:**
- 84岁老人新冠后彻夜失眠(温潜法)
- 智齿冠周炎牙龈红肿(潜阳丹)

### 复合[10] 温病 + 郁热 = 透热转气(升降散系列)

**病机:** 温邪内郁、气机不畅

## 六、五分类的核心论断(医林独啸斋)

### 论断[1] 五类不是对立,而是同一疾病的动态阶段
> "中医的道理是一以贯之的!不论是伤寒、温病还是内伤,

> ——破解误区明明热入营血

### 论断[2] 中风与伤寒的区别:汗之有无
> "太阳病提纲证:脉浮、头项强痛、恶寒。
> +汗出 = 中风(桂枝汤)
> +无汗 = 伤寒(麻黄汤)
> 这是临床最简单的辨别方法。"
> ——医学共识

### 论断[3] 湿温病的"如油入面"特性
> "古人形容湿温病是'如油入面、难分难解'。
> 火锅油溅衣服,洗好久都留印子。
> 这就是湿温病缠绵难愈的根由。"
> ——新冠复阳发热20余天伴肺炎

### 论断[4] 温病治法不忌辛温
> "我们用温热药物,是为了让气血流通起来。
> 除非是热邪猖狂、毫无郁结淤堵之象,这个时候你直接清解就可以了。
> 但凡有一丝郁结未能外发的表现,就应该使它发出来。"
> ——高烧39三天刀片嗓案

### 论断[5] 热病用白虎,腑实用承气
> "阳明气分热,用白虎汤辛寒清热;
> 腑实已成,用承气汤苦寒攻下。
> 二者不可混用。"
> ——医学共识

### 论断[6] 中风与伤寒传变规律
> "中风传变多化热(因为表虚、邪易深入);
> 伤寒传变多化寒(因为表实、邪被格拒)。
> 但这不是绝对的,还要看病人的底子。"
> ——医学共识
