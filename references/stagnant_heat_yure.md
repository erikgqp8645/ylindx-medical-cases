## 核心病机

**热邪内郁、气机不畅、不得宣发透达。**

四条郁热来源：
1. **食积化热**（最常见）：胃肠积滞腐败化热 → 中焦向上熏蒸上焦肺脏
2. **湿邪遏热**："水曰润下"，湿邪沉重粘滞，热被湿拽住走不掉
3. **气机郁滞**：三焦气机不畅 → 该升不升、该降不降
4. **寒包火/误治郁闭**：大温大补（防己黄芪汤+鸡鸣散）→ 把热邪关在里面

## 辨识要点：寒热错杂、本实标虚

**最关键两点：**
- 服补剂症状反而加重 —— 本实标虚的铁证
- 舌红苔厚 + 脉沉细滑数 —— 苔厚邪气实，细数热郁于内

## 宣透郁热法核心处方架构

**赵绍琴宣透郁热法 = 升降散 + 栀子豉汤 + 三仁汤 + 上焦宣痹汤 思路加减**

```
┌─ 升降散（透热外出主轴）────────────────┐
│  蝉蜕、僵蚕、片姜黄、大黄               │
│  轻症去掉姜黄、大黄                      │
│  郁金可代姜黄                            │
└──────────────────────────────────────────┘
                  ↓ +
┌─ 栀子豉汤（宣透胸膈郁热）──────────────┐
│  栀子、淡豆豉                            │
└──────────────────────────────────────────┘
                  ↓ +
┌─ 三仁汤（开湿邪、热邪的通道）───────────┐
│  苦杏仁、白蔻仁、生薏苡仁               │
│  + 石菖蒲、佩兰（芳香化湿浊）           │
└──────────────────────────────────────────┘
                  ↓ +
┌─ 焦三仙（治热之源 —— 食积）────────────┐
│  焦山楂、焦麦芽、焦神曲                  │
└──────────────────────────────────────────┘
                  ↓ +
┌─ 降肺化痰药（痰从大肠走）───────────────┐
│  枇杷叶、前胡、桑叶、全瓜蒌             │
│  肺与大肠相表里，痰降则大便滑           │
└──────────────────────────────────────────┘
```

## 关键心法

**"借力打力"原则（tsp南极）：**
> "热邪本就有向上向外漂浮的势头，所以就用辛凉药物，顺着这个势头就把体内的热邪从肌表透散出去。"
> —— 武术中的"借力打力"

**不要做的事：**
- ❌ 不要用大苦大寒的黄连去"逆势直折"
- ❌ 不要轻用大黄、生石膏（没到阳明病正证时）
- ❌ 不要大温大补（防己黄芪汤、鸡鸣散等）
- ❌ 不要强行止咳 —— 咳出来才是好事

## 服药后的"加重"现象

**宣透郁热后咳嗽反而加剧 —— 这是好事，不是坏事！**

原理：宣透后痰邪松动、往外排，肺里攒着的痰毒需要咳出来。**千万别止咳！越止越憋越深！**

正确做法：
- 拍背促进咳痰
- 让病邪有出路
- 坚持服完疗程

## 典型医案索引

### 小朋友积热咳嗽（赵绍琴宣透郁热法一剂愈）
- 11岁女孩，56斤，怕热，痰粘难咳出
- 舌：舌红，前部无苔起刺；舌根覆盖白腻苔
- 处方：蝉蜕6、炒僵蚕6、木蝴蝶3、栀子3、淡豆豉6、苦杏仁9、焦三仙各9、桑叶9、枇杷叶6
- 解析："升降散化裁（去姜黄、大黄，加木蝴蝶、桑叶）+ 栀子豉汤 + 苦杏仁枇杷叶 + 焦三仙"
- **一剂就好，第三剂都没吃**

### 小孩长期脚面浮肿（益气温阳利水无效，宣透郁热法治愈）
- 11岁孩子，从出生脚面浮肿，活动量大就肿
- 家长自开方：鸡鸣散+防己黄芪汤+千金苇茎汤合方 → 上火加重
- tsp南极辨证：**郁热不透** 为主导
- 处方：蝉蜕9、片姜黄6、僵蚕10、佩兰10、大腹皮10、槟榔10、焦三仙各10、川楝子6、瓜蒌15、桃仁15、芦根15、水红花子10、大黄5、滑石24、白术10、荆芥9、防风9

### 宣透郁热后咳嗽"加重"（不！那是快好了）
- 3岁多孩子，咳嗽3天，麻杏石甘汤+小儿消积止咳口服液无效

- 服后咳嗽加剧 → 拍背促进咳痰 → 坚持吃完大大减轻

## 相关方剂体系

### 升降散（杨栗山《伤寒温疫条辨》）
- **原方**：蝉蜕、僵蚕、片姜黄、大黄
- **轻症化裁**：去姜黄、大黄，加桑叶、木蝴蝶
- **变法**：郁金代姜黄
- "五根汤为升降散的升级版"（tsp南极）

### 栀子豉汤（张仲景《伤寒论》）
- 详见 references/gardenia_fermented_soybean_formula.md
- 在宣透郁热法中：宣透胸膈郁热

### 三仁汤（吴鞠通《温病条辨》）
- 苦杏仁、白蔻仁、生薏苡仁
- "开湿邪、热邪的通道"

### 千金苇茎汤（孙思邈）
- 清理肺中痰热
- 大便不通时冬瓜仁换全瓜蒌（肺与大肠相表里）

### 麻杏石甘汤（张仲景）
- 经方思维：麻黄辛散带石膏清上热，杏仁降肺
- 加桑白皮 = 日本汉方家常用的五虎汤

### 五根汤（tsp南极升级版升降散）
- 详见五根汤专题

### 消风和胃汤（胡天雄）
- 谷精草30g、鸡内金10g、五谷虫10g、煅牡蛎15g、党参10g
- 善后方：用于疳积、食积化热后调理

## 一句话核心心法

**「借力打力，透热外达；开湿通路，给邪出路。」**

治郁热的四步法：
1. 辛凉之品顺着热势往上往外透（升降散）
2. 化湿之药把阻挡热邪的湿邪去掉（三仁汤）
3. 消食之药掐断热邪的源头（焦三仙）
4. 降肺之药把痰浊从大肠导出（枇杷叶+全瓜蒌）
