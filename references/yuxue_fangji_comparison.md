## 一、通窍活血汤

[出处] 清·王清任《医林改错》
[方歌] 「**通窍全凭好麝香,桃仁大枣与葱姜;川芎黄酒赤芍药,表里通经第一方**」

### 1.1 组成与方解

```
        ┌──────────────┐
        │ 通窍活血汤  │
        └──────┬───────┘
               │
    ┌──────┬───┴────┬──────┐
    │      │        │      │
 赤芍  川芎      桃仁   红花  ← 活血化瘀(基础)
 葱    鲜姜     红枣    麝香  ← 通窍引经
              黄酒 ← 引经报使
```

### 1.2 主治(王清任原方 14 条症目)

  · 头发脱落
  · 眼疼白珠红
  · 糟鼻子
  · 耳聋年久
  · 白癜风
  · 紫癜风
  · 紫印脸
  · 青记脸如墨
  · 牙疳
  · 出气臭
  · 妇人干血劳
  · 男子劳病
  · 交节病作
  · 小儿疳证

### 1.3 核心指征

  · 病位:**头面 + 四肢 + 表层经络**
  · 病机:**瘀血阻窍(头面)**
  · 病程:年久(王清任强调「年久」)
  · 共性:病在头面部,病程长,常规治法无效

### 1.4 现代应用

  · 顽固性头痛(瘀血性)
  · 脑外伤后遗症
  · 脑震荡后遗
  · 脱发(瘀血型)
  · 过敏性鼻炎(部分)
  · 视网膜病变

### 1.5 注意事项

  · 麝香昂贵,现代可用人工麝香或冰片替代
  · 黄酒煎服,可助药力
  · 孕妇禁用

## 二、血府逐瘀汤 ★★★★★

[出处] 清·王清任《医林改错》
[方歌] 「**血府当归生地桃,红花甘草壳赤芍;柴胡芎桔牛膝等,血化下行不作痨**」

### 2.1 组成与方解

```
                  ┌──────────────────┐
                  │   血府逐瘀汤(11 味)│
                  └────────┬─────────┘
                           │
       ┌─────────┬─────────┼─────────┬─────────┐
       │         │         │         │         │
    活血化瘀  养血滋阴  行气开胸  引经报使  调和诸药
       │         │         │         │         │
   桃仁 红花   当归    柴胡 枳壳  桔梗 川芎   甘草
   赤芍       生地           牛膝(下行)
```

  · **君**:桃仁、红花(活血化瘀)
  · **臣**:当归、生地(养血滋阴,瘀去不伤正)
  · **佐**:柴胡、枳壳、桔梗(开胸行气,气行则血行)
  · **佐**:川芎、赤芍(助活血)
  · **使**:牛膝(引瘀血下行)、甘草(调和)

### 2.2 病机

  王清任原话:「**惟血府之血,瘀而不活,最难分别**」
  病位:**胸中(膈膜以上,心肺咽喉区域)**
  病机:**气滞血瘀**(因胸中气机不畅导致血瘀,或血瘀导致气滞)

### 2.3 核心指征(王清任原方 19 条症目)

  · **头痛**(无表证、无里证、无气虚、无痰饮,忽犯忽好,百方不效)
  · **胸痛**(固定不移,针刺样)
  · **胸不任物**(卧时胸上须压一物)
  · **胸任重物**(夜卧令仆妇坐于胸方睡)
  · **天亮出汗**(醒后出汗,名自汗)
  · **盗汗**
  · **灯笼病**(身外凉、心内热)
  · **呃逆**(饮水即呛)
  · **干呕**
  · **晚发一阵热**(每晚内热,兼皮肤热一时)
  · **瘀血性失眠**(夜不安、将卧则起,一夜无宁刻)
  · **瘀血性多梦**(王清任主张用此方)
  · **瘀血性咳嗽**(侧卧一边,翻身则咳益甚)
  · **瘀血性不孕**(行气理血)
  · **假孕**(经闭腹大,下其瘀血则胀自消)
  · **小儿痞块**
  · **妇人干血劳**
  · 肝郁血瘀(扁平疣)
  · 眼底出血(视网膜中央静脉阻塞)

### 2.4 加减化裁(中医世家 67 卡片)

| 兼证 | 加味 |
|---|---|
| 气郁 | 香附、薤白 |
| 血瘀甚 | 丹参、三七(冲服) |
| 痰盛 | 旋覆花 |
| 便秘 | 大黄(后下) |
| 发热 | 生石膏、黄芩(加重柴胡) |
| 久病气虚 | 黄芪(30g) |
| 年高 | 山萸肉 |
| 眼底水肿 | 琥珀、车前子、猪茯苓 |
| 硬性渗出 | 昆布、海藻、牡蛎、鳖甲 |
| 血脂高 | 决明子、山楂、郁金、菊花、蒲黄 |

### 2.5 现代临床应用

  · 瘀血性头痛、偏头痛
  · 冠心病、心绞痛(瘀血型)
  · 瘀血性失眠
  · 哮喘、肺心病(合大柴胡 / 桂枝茯苓丸)
  · 眼底出血、糖尿病视网膜病变
  · 颅脑外伤后遗症
  · 瘀血性发热(小儿外伤后持续发热)
  · 妇科不孕(瘀血型)
  · 银屑病、痤疮、慢性荨麻疹(合清热解毒药)

  · 14 帖头痛得愈

  · 处方:血府逐瘀汤每日 1 剂

## 三、膈下逐瘀汤

[出处] 清·王清任《医林改错》
[方歌] 「**膈下逐瘀桃牡丹,赤芍乌药玄胡甘;川芎灵脂红花壳,香附开郁血亦安**」

### 3.1 组成

  桃仁 丹皮 赤芍 乌药 元胡 甘草 川芎 灵脂 红花 枳壳 香附

### 3.2 病机与病位

  · 病位:**膈下(肚腹,肝脾区域)**
  · 病机:**瘀血痞积**(气滞 + 血瘀,痞块形成)
  · 与血府逐瘀区别:血府逐瘀偏气滞,膈下逐瘀偏痞积(有形积块)

### 3.3 主治

  · 积块(肚腹)
  · 小儿痞块、肚大青筋
  · 痛不移处
  · 卧则腹坠
  · 肾泻(五更泻)
  · 久泻
  · 痛经(气滞血瘀型)
  · 闭经(气滞血瘀)
  · 血瘀性不孕(情志所伤,兼见胸胁胀满)

### 3.4 核心指征(中医世家)

  · **少腹胀痛**(排气得舒 = 气滞)
  · 经血色紫暗有块
  · 经前腹痛拒按
  · 偏于气滞者,重用乌药、延胡索、枳壳、香附
  · 与少腹逐瘀汤区别:膈下逐瘀治气滞重,少腹逐瘀治寒凝重

## 四、少腹逐瘀汤 ★★★★

[出处] 清·王清任《医林改错》
[方歌] 「**少腹茴香与炒姜,元胡灵脂没芎当,蒲黄官桂赤芍药,种子安胎第一方**」

### 4.1 组成与方解

```
              ┌──────────────────┐
              │   少腹逐瘀汤      │
              └────────┬─────────┘
                       │
       ┌─────────┬─────┴─────┬─────────┐
       │         │           │         │
    温经散寒    活血化瘀    行气止痛  养血活血
       │         │           │         │
   小茴香 干姜   蒲黄 五灵脂  元胡 没药   当归 川芎
    官桂         赤芍                  (温而不燥)
```

  · 特点:**温经散寒 + 活血化瘀 + 行气止痛**三法并用
  · 大量温热药(小茴香、干姜、官桂),故偏温,适合寒凝证

### 4.2 病机与病位

  · 病位:**少腹(下焦)**
  · 病机:**寒凝血瘀**(经期产后感寒,寒凝血瘀)
  · 与血府逐瘀区别:血府逐瘀偏气滞血瘀(凉/平),少腹逐瘀偏寒凝血瘀(温)

### 4.3 主治(王清任原方 + 中医世家)

  · 少腹积块疼痛
  · 或有积块不疼痛
  · 或疼痛而无积块
  · 少腹胀满
  · 经血见时,先腰酸少腹胀
  · 经血一月见三、五次,接连不断,断而又来
  · 经色紫黑或块
  · 崩漏
  · 兼少腹疼痛
  · 粉红兼白带
  · **种子**(王清任自述:「此方种子如神,每经初见之日吃起,一连吃五付,不过四月必成胎」)
  · **安胎**(去疾、种子、安胎,「尽善尽美」)
  · 习惯性流产(子宫内瘀血占位)

### 4.4 核心指征(中医世家 19 卡片)

  · **小腹冷痛**(经血色暗有块,喜温拒按,血块下后痛减)
  · 经期错后
  · 月经过少(寒凝血瘀型)
  · 痛经(寒凝型)
  · 不孕症(感受寒邪型)
  · 闭经(寒凝血瘀型)
  · 血疝(少腹硬满,《医宗金鉴》)
  · 湿疸「壮热,少腹痞满、疼痛,内有瘀血」(《中医名词术语》)
  · 精液不液化(阳虚寒盛型,有效率 96%)

### 4.5 加减化裁

  · 偏气滞(痛胀):加香附、乌药
  · 偏寒凝(冷痛):重用官桂、炮姜,加肉桂
  · 虚寒明显:加鹿角胶、紫河车
  · 寒湿带下:加苍术、薏苡仁

## 五、身痛逐瘀汤 ★★★

[出处] 清·王清任《医林改错》
[方歌] 「**身痛逐瘀膝地龙,羌秦香附草归芎,黄芪苍柏量加减,要紧五灵桃没红**」

### 5.1 组成

  秦艽1钱 川芎2钱 桃仁3钱 红花3钱 甘草2钱 羌活1钱
  没药2钱 当归3钱 灵脂2钱炒 香附1钱 牛膝3钱 地龙2钱去土

### 5.2 病机与病位

  · 病位:**全身经络**(肩、臂、腰、腿、周身)
  · 病机:**瘀血入络(瘀血痹)**

### 5.3 王清任原方论断(极其重要)

  > 「**凡肩痛、臂痛、腰痛、腿痛、或周身疼痛,总名曰痹症。**
  > 明知受风寒,用温热发散药不愈;明知有湿热,用利湿降火药无功;
  > 久而肌肉消瘦,议论阴亏,随用滋阴药又不效。
  > **因不思风寒湿热入皮肤何处作痛,入于气管,痛必流走;入于血管,痛不移处。**
  > 如论虚弱是因病致虚,非因虚而致病。
  > **总滋阴,外受之邪归于何处?总逐风寒,去湿热,已凝之血,更不能活。**
  > 如水遇风寒,凝结成冰,冰成风寒已散。明此义,治痹症何难。」

### 5.4 核心指征

  · **痛不移处**(关键鉴别:游走痛为风寒,固定痛为瘀血)
  · 久治不愈(各种祛风散寒除湿滋阴都无效)
  · 舌质紫、舌下静脉曲张
  · 腹诊:受伤侧压痛点、肌肉高出
  · 加减:微热加苍术、黄柏;虚弱量加黄芪一二两

### 5.5 真实医案

[医林独啸斋] 72 岁老翁腰部屈伸不利 15 年:
  · 处方:**桂枝茯苓丸合身痛逐瘀汤 + 生白芍 30g**
    桂枝15 茯苓15 赤芍15 桃仁15 丹皮15 秦艽9 川芎6 红花9
    炙甘草6 羌活6 制没药6 当归9 炒五灵脂9 香附9 怀牛膝30 地龙9
    生白芍30
  · 反馈:**才两剂就见效**「腰基本不难受了」、「腰部感觉活络起来了」
  · 瞑眩:腹泻 5-6 次/日(芍药动大便 + 祛瘀血从大便排出)

[大塚敬节] 31 岁男子投球后腰背痛:
  · 处方:桂枝茯苓丸
  · 反馈:仅 7 天已无疼痛,其后不再发而好了

## 六、补阳还五汤

[出处] 清·王清任《医林改错》
[方歌] 「**补阳还五耆归芎,桃红赤芍加地龙;半身不遂中风证,益气活血经络通**」

### 6.1 组成(独重黄芪)

```
       ┌────────────┐
       │ 补阳还五汤 │
       └─────┬──────┘
             │
   ┌─────────┼─────────┐
   │         │         │
 重用补气  活血化瘀  活血通络
   │         │         │
 黄芪4两   当归尾2钱  地龙1钱
 (最大特色) 赤芍1.5钱  (引经通络)
           川芎1钱
           桃仁1钱

```

  · ★ 黄芪**重用 4 两**(补气以活血,最大特色)
  · 当归尾 2 钱,赤芍 1.5 钱
  · 桃仁、红花、地龙、川芎各 1 钱

### 6.2 病机与病位

  · 病位:**半身**(脑卒中后遗)
  · 病机:**气虚血瘀**(元气亏损,血络瘀阻)

### 6.3 主治

  · 半身不遂
  · 口眼歪斜
  · 语言蹇涩
  · 口角流涎
  · 大便干燥
  · 小便频数、遗尿不禁
  · 中风后遗

### 6.4 用法要点

  · 「不是剂量大小的问题,而是层次的问题」

### 12.4 久痛入络(王清任论)

  > 「**入于血管,痛不移处。如论虚弱是因病致虚,非因虚而致病**」

### 12.5 祛瘀与腹泻(瞑眩反应)

  > 「**服用祛瘀血药物的患者,如果出现腹泻,往往恢复迅速。
  > 这是因为借助肠道这个排邪通道,把体内的瘀血从大便一举排出之故**」

### 12.6 守方的必要性(慢性病)

  > 「**治疗急性病要有胆有识,治疗慢性病要有方有守**」——岳美中

### 12.7 重用生白芍的「动大便」原理

  > 「**一切可以补血、补阴的药物,重用之后都可以让大便通畅,甚至于腹泻。
  > 芍药可以解除肌肉的痉挛,肠道肌肉吸收了芍药的药力之后,
  > 就会变得放松、柔软,导致人变成'直肠子'**」

### 12.8 大塚敬节经验

  > 「**日本汉方医家遇到腹诊出淤血指征的患者,不论病名如何,
  > 均根据患者体质虚实选取桂枝茯苓丸、当归芍药散、桃核承气汤
  > 等祛瘀血处方,往往有神效**」

## 第十四部分:核心鉴别决策树

### 14.1 第一步:辨病位

```
  头面四肢 → 通窍活血汤
  胸中 → 血府逐瘀汤
  膈下肚腹 → 膈下逐瘀汤
  少腹冷痛 → 少腹逐瘀汤
  全身经络痛 → 身痛逐瘀汤
  半身不遂 → 补阳还五汤
  下焦实证 → 桂枝茯苓丸
  下焦血虚 → 当归芍药散
  下焦急症 → 桃核承气汤
  通用瘀痛 → 失笑散
  干血死血 → 下瘀血汤
```

### 14.2 第二步:辨病机寒热虚实

```
  气滞血瘀(平/凉) → 血府逐瘀汤
  寒凝血瘀(温) → 少腹逐瘀汤
  瘀热互结(寒) → 桃核承气汤
  血虚瘀血(平/温) → 当归芍药散
  实证瘀血(平) → 桂枝茯苓丸
  气虚血瘀(温) → 补阳还五汤
  瘀血入络(平) → 身痛逐瘀汤
  干血死血(峻) → 下瘀血汤 / 虫类
```

### 14.3 第三步:辨强度

```
  峻攻(实证急症) → 桃核承气汤 / 下瘀血汤 / 复元活血汤
  中攻(普通瘀血) → 桂枝茯苓丸 / 膈下逐瘀汤 / 身痛逐瘀汤
  缓攻(体质较弱) → 失笑散 / 少腹逐瘀汤
  攻补兼施 → 血府逐瘀汤(养血+行气)
  补中寓攻 → 当归芍药散 / 补阳还五汤
```

### 14.4 第四步:死血加强(虫类)

```
  瘀血日久 → 加水蛭 6-12g / 土鳖虫 6-12g
  「患者一下子就觉得患处轻松了,腹诊压痛也消失了」
```

### 14.5 第五步:合并他证(常合方)

```
  瘀血 + 胸胁苦满 → 加柴胡剂(小/大柴胡)
  瘀血 + 寒 → 加肉桂、干姜、附子
  瘀血 + 虚 → 加黄芪、当归、党参
  瘀血 + 痰湿 → 加半夏、南星、苍术(二朮汤思路)
  瘀血 + 出血 → 加三七、茜草(祛瘀止血)
  瘀血 + 痛剧 → 加乳香、没药、延胡索
```
